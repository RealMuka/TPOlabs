# Отчёт по тестированию ПО — Веб-сайт “Kufar.by”

## 1. Введение

**Цель работы:**  
Провести системное тестирование веб-сайта “Kufar.by” для проверки корректности работы функционала, выявления дефектов и оценки соответствия требованиям.

**Краткое описание программы:**  
Сайт доски объявлений позволяет пользователям размещать, редактировать, удалять объявления, искать по категориям/ключевым словам, применять фильтры (цена, регион, состояние) и просматривать список объявлений.  
Ссылка на программу/сайт/приложение: [kufar.by](https://www.kufar.by/)

**Обоснование тестирования:**  
Системное тестирование необходимо для оценки поведения сайта с точки зрения конечного пользователя, проверки соответствия требованиям (SRS), интерфейса и производительности.

---

## 2. Тест-план (Test Plan)

## 2.1 Цели и задачи

- Проверка корректности добавления, редактирования и удаления объявлений
- Проверка работы поиска и фильтров
- Проверка корректного отображения элементов интерфейса
- Выявление ошибок и дефектов
- Проверка соответствия требованиям спецификации

## 2.2 Область применения

**Тестируемые модули:**

- Модуль размещения объявлений
- Модуль редактирования объявлений
- Модуль удаления объявлений
- Модуль отметки объявлений как "Поднять"
- Модуль поиска и фильтрации

**Ограничения:**

- Тестирование проводится только на десктопной версии в Chrome
- Не тестируются мобильные версии, регистрация/авторизация и альтернативные браузеры

## 2.3 Стратегия тестирования

- Функциональное тестирование всех модулей
- Регрессионное тестирование после исправления багов
- Тестирование GUI для проверки корректного отображения интерфейса
- Smoke-тестирование при каждом запуске сайта

## 2.4 Критерии начала тестирования

- Сайт открывается в браузере без ошибок
- Реализован основной функционал (CRUD операций с объявлениями)

## 2.5 Критерии завершения тестирования

- Выполнено ≥ 90% тест-кейсов
- Все критические баги зарегистрированы и описаны
- Сайт соответствует требованиям SRS в основной функциональности

## 2.6 Тестовая среда

- **ОС:** Windows 10 Pro (64-bit)
- **Браузер:** Chrome v112
- **Сервер:** продакшн-сервер Kufar.by
- **Дополнительно:** интернет-соединение, редактор кода, утилита для снятия скриншотов
